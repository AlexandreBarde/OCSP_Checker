# Pour webpack

Dans le dossier Firefox/

Après avoir installé NodeJS

1. Installer les dépendances:
```
npm install
```

2. Installer webpack

```
sudo npm install webpack -g
```

3. Compiler les \*.bundle.js:
```
webpack
```

4. Compiler à chaque modification
```
webpack --watch
```
