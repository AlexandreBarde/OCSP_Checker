# Compilation
1. Créer un dossier build/
2. Se placer dans ce dossier et faire:
```
cmake .. && make
```

# Pour webpack

Dans le dossier Firefox/

1. Installer les dépendances:
```
npm install
```

2. Compiler les \*.bundle.js:
```
webpack
```

3. Compiler à chaque modification
```
webpack --watch
```
